Risk Guard Circuit Breakers

Three independent checks run before every trade. The first failure halts execution and fires a Telegram alert.

1. Balance Floor ($5)
Blocks if USDC balance drops below the minimum. Prevents trading into zero.
e.g. balance = $4.20 → BLOCKED
2. Session Drawdown (10%)
Blocks if confirmed lost trades in the last 24h exceed 10% of the session start balance. Only real losses count — in-flight positions are excluded.
e.g. start=$20, losses=$2.10 → 10.5% → BLOCKED
3. Consecutive Losses (3)
Blocks if the last 3 resolved trades are all lost. Signals a systematic issue requiring manual review.
e.g. lost, lost, lost → BLOCKED
